Notification Image Size Guide for Android and iOS

This guide provides the recommended image sizes for push notifications on both Android and iOS devices. It covers small icons, large icons, and images for rich notifications to ensure compatibility across various devices.

Android Image Sizes

Type MDPI HDPI XHDPI XXHDPI XXXHDPI Format
Small Icon (Status Bar) 24x24 px 36x36 px 48x48 px 72x72 px 96x96 px PNG (Transparent)
Large Icon (Notification) 48x48 px 72x72 px 96x96 px 144x144 px 192x192 px PNG (Transparent)
Big Picture (Expanded View) Recommended: 1200x600 px JPG or PNG

iOS Image Sizes

Type 2x Size 3x Size Aspect Ratio Format
Notification Icon (App Icon) 40x40 px 60x60 px 1:1 PNG
Rich Notification Image 1080x1080 px 1280x720 px 1:1 or 2:1 JPG or PNG
